# Kalqix MM (Multi-Wallet Market Maker)

Encrypted, multi-wallet market maker runner for Kalqix.

## Features
- Run unlimited wallets
- Encrypted API keys & seeds
- One-command install
- Parallel execution
- Clean shutdown
- Beginner friendly

## Installation

```bash
git clone https://github.com/CryptoThrone/Kalqix-mm.git
cd Kalqix-mm
./install.sh
